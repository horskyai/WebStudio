# WebStudio – Pravidla pro Claude

## Design z katalogu

Při navrhování jakékoli webové stránky nebo UI vždy čerpej design z katalogu a jeho kategorií:

- **Buttony** → `buttony/`
- **Hero sekce** → `hero/`
- **Navigace** → `navigace/`
- **Loadery** → `loadery/`
- **Intra** → `intros/`
- **Mikrointerakce** → `mikrointerakce/`
- **Drafty / rozepsané věci** → `drafty/`
- **Přehled katalogu** → `katalog.html`

Nikdy nevymýšlej nový standalone design od nuly. Vždy procházej existující kategorie v katalogu a skládej stránky z komponent, které tam jsou. Pokud chybí komponenta, přidej ji nejprve do katalogu a příslušné složky.
